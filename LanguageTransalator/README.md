# AI-Powered Language Translation Tool

## Description

This project is a GUI-based Language Translation Tool developed using Python. It allows users to translate text into multiple languages and listen to the translated output using Text-to-Speech technology.

## Features

* Translate text into multiple languages
* Automatic source language detection
* User-friendly GUI
* Copy translated text
* Text-to-Speech support
* Error handling

## Technologies Used

* Python
* Tkinter
* Deep Translator
* gTTS (Google Text-to-Speech)

## Supported Languages

* Tamil
* Hindi
* French
* German
* Spanish
* Japanese
* Korean
* Chinese
* Arabic
* Russian

## Installation

Install dependencies:

pip install -r requirements.txt

## Run Project

python translator_gui.py
